Résolution des problèmes

Erreur 404 ou "Not Found"

Cette erreur signifie que le cache de votre navigateur contient des anciennes versions de fichiers.

Solution:

  • Cliquez sur
  • Suivez les 3 étapes de la page
  • Reconnectez-vous

Erreur 401 ou "Unauthorized"

Cette erreur signifie que votre session a expiré ou que votre token est invalide.

Solution:

  • Déconnectez-vous de l'application
  • Reconnectez-vous avec vos identifiants

Écran blanc ou application ne charge pas

Solutions:

  • Fermez complètement le navigateur et rouvrez-le
  • Videz le cache du navigateur:
    • iPhone: Réglages → Safari → Effacer historique et données
    • Android: Chrome → Paramètres → Confidentialité → Effacer les données
  • Utilisez la page

Les photos ne s'affichent pas

Causes possibles:

  • Connexion internet lente ou instable
  • Permissions d'accès aux fichiers

Solution: Rechargez la page en tirant vers le bas

Outils de diagnostic

Support

En cas de problème persistant:

  • Vérifiez que vous utilisez la dernière version de votre navigateur
  • Essayez avec un autre navigateur (Chrome, Safari, Firefox)
  • Contactez votre administrateur avec une capture d'écran de l'erreur
