Forcer le rechargement complet

Si vous rencontrez des erreurs 404 ou si l'application ne charge pas correctement, utilisez cette page pour forcer un rechargement complet.

Étape 1: Désactiver le Service Worker
Supprime le Service Worker qui peut cacher des anciennes versions
Étape 2: Vider tout le cache
Efface tous les fichiers mis en cache par l'application
Étape 3: Effacer le token
Supprime le token d'authentification (vous devrez vous reconnecter)
